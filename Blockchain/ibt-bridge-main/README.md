# IBT Bridge Project

Cross-chain bridge between Ethereum (Sepolia) and Sui (Testnet)

## Components
- ethereum-contracts/ - Solidity smart contract
- sui-contracts/ - Move smart contract  
- bridge-backend/ - Node.js bridge service
- bridge-frontend/ - React web app

## Testnets
- Ethereum: Localnet
- Sui: Localnet
